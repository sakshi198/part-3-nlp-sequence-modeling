# Part 3 – NLP Sequence Modeling

## Project Overview
This project demonstrates a complete NLP pipeline for customer support text classification. The notebook covers dataset understanding, text preprocessing, TF-IDF vectorization, baseline modeling using Logistic Regression, sequence modeling using LSTM, and conceptual discussion of attention and transformers.

## Dataset
- customer_support_text_classification.csv
- data_dictionary.md

## Tasks Completed
1. Dataset Understanding
2. Text Preprocessing
3. Text Vectorization
4. Baseline Model
5. Sequence Model (LSTM)
6. Attention and Transformer Reflection
